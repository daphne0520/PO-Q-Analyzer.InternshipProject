Project Flow — Overview
Report Approval & Verification System — Low-Fidelity Wireframe Set
01
Approval Portal
Search & review reports. Approve or Require Amendment. Confirms email notification and moves to Executive Dashboard.
02
Verification Portal
AI Processor compares PO data. Amend or generate/preview AI report, then submit — updates the po-reports table.
03
Executive Dashboard
Overview cards (Awaiting Verify / Approval / AI Issued), search & preview a report, download or send for amendment.
End-to-end flow
  • Executive Dashboard → select report → Verification Portal
  • Verification Portal → AI comparison → Amend or Generate AI Report → submit → back to Executive Dashboard
  • Executive Dashboard / queue → Approval Portal → Approve or Require Amendment → email notification → Executive Dashboard
